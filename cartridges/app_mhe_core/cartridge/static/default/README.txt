Install node.js
http://nodejs.org/download/

Install Ruby (1.9.3)
http://rubyinstaller.org/downloads/

Install compass
type 'gem install compass' into command prompt one ruby is installed

Install grunt
http://www.ghosthorses.co.uk/production-diary/installing-grunt-on-os-x-and-windows-7/#grunt_win7
npm install -g grunt-cli

to install all project dependencies, first navigate to the project directory in command prompt (where the gruntfile is located), and type
npm install

to compile compass you can then run:
grunt dev
